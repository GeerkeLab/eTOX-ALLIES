Automated Line for Linear Interaction Energy-based Screenings

eTOX ALLIES is an integrated tool that easily allows binding affinity prediction of small molecule toward proteins by the iterative Linear Interaction Energy method.

How to start:
- Screen a set of compounds: go to SUBMIT page. Select the model protein and version and upload your SDF file.

- Create a model for a new protein: go to New Model in the MODELS page. Define the properties for the new iLIE model, providing the already optimized (e.g. no missing atoms/ residued, minimized, flipped sidechains) PDB file(s) template for the protein.

- Calibrate a protein model for a training-set of compounds: go to SUBMIT. Select Type of Calculation : CALIBRATION. Be sure the SDF file of the training set contains a specific field reporting the reference DGbind value.
